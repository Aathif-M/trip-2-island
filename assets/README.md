# Media Assets Guide

To replace the images and videos on the website, simply place your media files in this `public/assets/` folder. **You must name the files exactly as listed below** for the website to automatically detect and load them.

> **Note:** The code is currently set up to look for `.jpg` for images and `.mp4` for the video.

## Home Page - Hero Section
*   **`hero-bg.mp4`**: A sweeping, cinematic video of Sri Lanka landscapes to automatically loop in the background of the Home page hero section.
*   **`hero-bg.jpg`**: A fallback image for the Home page hero section (shown while the video loads, on devices that restrict autoplay, or if there is no video file present).

## Home Page - Featured Stays
*   **`stay-wild-coast.jpg`**: Image representing "Wild Coast Tented Lodge" (Jungle luxury).
*   **`stay-amangalla.jpg`**: Image representing "Amangalla" (Boutique Heritage in Galle Fort).
*   **`stay-tea-trails.jpg`**: Image representing "Ceylon Tea Trails" (Tea estate bungalows).

## Home Page - Regions Grid
*   **`region-south-coast.jpg`**: Golden beaches, palm trees, and surf towns (South Coast).
*   **`region-cultural-triangle.jpg`**: Ancient ruins and sacred temples (Cultural Triangle).
*   **`region-hill-country.jpg`**: Misty tea estates and waterfalls (Hill Country).
*   **`region-northern-explorer.jpg`**: Untamed nature, beaches, and vibrant heritage (Northern Explorer).

## Home Page - Curated Itineraries
*   **`itinerary-culture.jpg`**: Image representing "Culture & History" (e.g., temples, ancient cities, cultural experiences).
*   **`itinerary-wildlife.jpg`**: Image representing "Wildlife Safari" (e.g., elephants, leopards, safaris).
*   **`itinerary-honeymoon.jpg`**: Image representing "Honeymoon Escapes" (e.g., romantic beach dinner, secluded luxury).
*   **`itinerary-family.jpg`**: Image representing "Family Adventures" (e.g., engaging activities, family fun, elephants).

## Inner Pages (Hero/Banner Images)
*   **`page-places-to-stay.jpg`**: A stunning, inviting hero image for the top of the "Places To Stay" page (e.g., a beautiful villa or lodge).
*   **`page-journal.jpg`**: A captivating hero image for the "Travel Journal" page (e.g., someone exploring, scenic view).
*   **`page-itineraries.jpg`**: A sweeping hero image for the "Curated Itineraries" page.
*   **`page-contact.jpg`**: A welcoming side image for the Contact page (e.g., a warm greeting, a local smile, or a beautiful beach view).
*   **`page-about.jpg`**: The main hero background image for the About page (e.g., a local guide or an immersive local experience).
*   **`map-sri-lanka.jpg`**: A visually appealing map or stylized overview of Sri Lanka (used in the Destinations page).
*   **`region-east-coast.jpg`**: Unspoiled white-sand beaches (East Coast, used in the Destinations page regions list).
